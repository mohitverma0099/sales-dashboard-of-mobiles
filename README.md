Mobile Sales Analysis Dashboard 📱📊
Project Overview

This project focuses on analyzing mobile phone sales data to identify sales trends, customer purchasing behavior, brand performance, and overall business growth using Excel, Python, and Power BI.

The dashboard helps businesses understand:

yearly sales trends
Best-selling mobile brands
Top-performing products
Regional sales performance
Which payment method is best
Revenue by top cities

This project demonstrates an end-to-end data analytics workflow from data collection to dashboard creation.
